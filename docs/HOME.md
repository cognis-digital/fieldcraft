# fieldcraft - Modern Soldier OS

> **Disclaimer:** Educational/analytical/defensive material. Not operational instructions for violence or weapons.

A practical, wiki-style readiness OS for soldiers and civilians: survival, food/water, field medicine, rucking, navigation, comms, gear, preparedness, plus modern-conflict lessons, camouflage/concealment, night, CBRN-defensive, tactical fitness, field hygiene, SOPs, an acronym glossary, and a role/loadout builder. Query it with the `fieldcraft` CLI.
